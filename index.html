Om kursen

Progressiv inlärning — från HTML till JavaScript

Webutveckling 1 är upplagd som en progressiv resa där varje modul bygger vidare på föregående. Du börjar med ren HTML-struktur och tillgänglighet, lägger sedan till CSS för utseende och layout, och avslutar med JavaScript för interaktivitet.

CSS-användningen i kursmaterialet följer samma progression: de tidiga modulerna använder enkel inline-stil och grundläggande selektorer, medan de senare modulerna introducerar flexbox, responsiv design, animationer och CSS-variabler — precis som du lär dig dem i kursen.

Projektarbetet i modul 7 ger dig möjlighet att kombinera alla tekniker i ett eget projekt. Modul 8 behandlar de juridiska och etiska ramarna som gäller för all webbutveckling.
